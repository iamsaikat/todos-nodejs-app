# ts-node-todo

`ToDo App` written in NodeJS with TypeScript and MongoDB

## Build with

- Node Js, Express, mongoose
- nodemon, prettier, typescript

## Features

- Add todo
- Toggle todo
- Update todo
- Remove todo
- Clear all completed todos

Start app in development mode:

> npm run dev

Start app in production mode

> npm start

# Technology Stack

* Node `v16.14.0`
* Npm `8.5.3`
* Express `4.17.17`
* Mongoose `7.2.0`
* winston `3.8.2`
